# Enterprise AI IT Support Assistant

Enterprise-grade hybrid AI assistant for IT troubleshooting using semantic search and conversational AI.

Features:
- Hybrid AI architecture
- Knowledge base retrieval
- Conversational AI using Ollama
- Enterprise UI dashboard
- System monitoring
- Real-time streaming responses

Tech stack:
- Python
- Streamlit
- SentenceTransformers
- Ollama LLM
- Semantic Search
